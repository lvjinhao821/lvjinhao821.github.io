基本概念

3.1语法

1.JavaScript（跟风取的名）的核心语言特性在ECMA-262（标准）中是以名为ECMAScript（依据前面标准实现的语言）的伪语言的形式来定义的。

2.ECMAScript中的一切(变量、函数名和操作符)都区分大小写。

3.标识符，就是指变量的、函数、属性的名字，或者函数的参数。由数字、变量、下划线和美元符$组成，数字不能开头。推荐采用驼峰命名法，不能把关键字、保留字、true、false和null作为标识符。

4.单行注释：两个斜杠//，块级注释：一个斜杠和一个星号开头，一个星号一个斜杠结尾， /*…..*/。

5.严格模式为这门语言中容易出错的地方施加了限制，用“use strict”写在顶部就行。

6.语句结束如果省略分号，会由解析器确定语句结尾。推荐语句结束后加上分号；。

3.3变量

7.ECMAScript的变量是松散类型的可以用来保存任何类型的数据，换句话说，每个变量仅仅是一个用于保存值的占位符而已。未经初始化的变量，会保存为undefined。var操作符定义的变量将成为定义该变量的作用域中的局部变量；如果定义变量时省略了var，则该变量成为全局变量，不推荐。定义多个变量用逗号,进行分隔。

3.4数据类型

8.ECMAScript中有5中基本数据类型（Underfined、Null、Boolean、Number和String）还有一种复杂数据类型Object，Object本质上是由一组无序的名值对组成的。

9.typeof可检测给定变量的数据类型（基本数据类型），instanceof用来检测给定变量是否是复杂数据类型。

alert(typeof str)   //string
alert([1,2] instanceof Array)   //true

10.对未初始化（没有赋值）和未声明（没有定义）的变量执行typeof操作符都会返回undefined。如果养成习惯每当定义变量的时候都初始化一个值，这样用typeof判断变量类型时如果返回了undefined，就能确定这个变量未声明。

11.null值表示一个空对象指针，这也正是使用typeof操作符检测null值时会返回“object”的原因。只要意在保存对象的变量还没有真正保存对象，就应该明确地让该变量保存null值。

12.Boolean类型只有两个字面值：true和false。ECMAScript中所有类型的值都有与这两个Boolean值等价的值（false、‘‘ ’‘空字符串、0、NaN、null、undefine都对应false）,可以对任何数据类型的值调用Boolean()函数，总会返回一个Boolean值。

var message = "Hello!"
var messageAsBoolean = Boolean(message);  //true

13.ECMAScript没有为整数和浮点数值分别定义不同的数据类型，Number类型可用于表示所有类型。数值字面量格式默认为十进制，如果想用八进制表示第一位必须是0，然后是八进制数字序列(0-7)；如果想用十六进制表示，第一位必须是0x，然后是十六进制数字(0-9)和(A+F)。在进行算术计算时，所有八进制和十六进制表示的数值最终都将转换成十进制。

var intNum = 55;        //整数
var octalNum1 = 070；   //八进制的56
var hexNum1 = 0xA;     //十六进制的10

  • 浮点数值—该数值中必须包含一个小数点，并且小数点后面必须至少有一位数字。

    • 由于保存浮点数值需要的内存空间是保存整数值的两倍，因此ECMAScript会不失时机地将浮点数值转换为整数（如：1.转换为1，10.0转换为10）。
    • 对于极大和极小的数值，可以用e表示法(科学计数法)表示浮点数值，3.125e7代表3.125乘以10的7次方。
    • 浮点数值进行算术计算时其精度远不如整数，因此可以先转换成整数，计算完后转换回浮点数。

  • NaN，即非数值，表示一个本来要返回数值的操作数未返回数值的情况。

    • 任何涉及NaN的操作(例如：NaN/10)都会返回NaN。

    • NaN与任何值都不相等，包括NaN本身。

    • isNaN()函数，可帮助我们确定这个参数是否 “不是数值”。isNaN()接收到一个值后，会尝试将这个值转换为数值，不能转换为数值的值会导致这个函数返回true。

      alert(isNaN(NaN));       //true
alert(isNaN(10));        //false 10是一个数值
alert(isNaN("10"));      //false 10是一个数值
alert(isNaN("blue"));    //true (不能转化成数值)

  • 将非数值转换为数值

    • Number()可用于任何数据类型

  • 具体转换规则看书p30

  • parseInt()专门把字符串转换成数值

    • 会忽略字符串前面的空格，直至找到第一个非空格字符。如果第一个字符不是数字或负号，则返回NaN；如果第一个字符是数字字符，则会继续解析，直到遇到非数字字符。小数点不是有效数字字符，遇到小数点也会停止解析。该函数能识别出各种整数格式，按照对应的格式解析数据。

    • 在解析像八进制字面量的字符串时，ECMAScript3和5存在分歧，为了消除分歧，可以为这个函数设置第二个参数：转换时使用的基数。(推荐设置)

      var num1 = parseInt("1234blue", 10);    //1234
var num2 = parseInt("", 10);            //NaN
var num3 = parseInt("22.5", 10);        //22
var num4 = parseInt("0xA", 16);         //10(十六进制数)

  • parseFloat()专门把字符串转换成数值

    • 与parseInt()一样，但遇到的第一个小数点是有效的，第二个才无效。它只能解析十进制，因为他会始终忽略前面的0。

      var num1 = parseInt("22.34.5", 10);    //22.34
var num1 = parseInt("0908.5", 10);     //908.5
var num1 = parseInt("3.125e7", 10);    //31250000

14.String类型用于表示由零或多个16位Unicode字符组成的字符序列，包含一些特殊的字符字面量(见p33).任何字符串的长度都可以通过length属性取得。ECMAScript中的字符是不可变的，也就是说字符一旦创建，他们的值就不能改变了，参照下图理解。(简单类型都是不可变的，复杂类型可变)

将值转换为字符串；

  • 使用几乎每个值都有的toString()方法，null和undefined没有这个方法。多数情况下都不必传递参数，但调用数值的toScring()方法时，可传递一个参数：输出数值的参数。

    var found = true;
alert(found.toString())    //"true"
var num = 11;
alert(num.toString(2))     //"1010" 转换为二进制，字符

  • 在不知道转换的值是不是null或undefined时，可以使用转型函数String()，这个函数能够将任何类型的值转换为字符串

    • 如果值有toString()方法，则调用该方法(没有参数)并返回相应结果。
    • 如果值是null,则返回“null”。
    • 如果值是undefined则返回“undefined”.

  • 要把某个值变为字符串，可以使用加号操作符，把他与一个字符串（“”）加在一起。

15.ECMAScript中的对象其实就是一组数据和功能的集合。要理解，在ECMAScript中，Object类型是所有它的实例的基础。换句话说Object类型所具有的任何属性和方法同样存在于更具体的对象中。Object的每个实例都具有下列属性和方法（p35）。

3.5操作符

16.只能操作一个值的叫做一元操作符

  • ++i先变后用，i++先用后变，–i和i–也同理。不仅适用于整数，也可用于字符串，布尔值，浮点数值和对象，规则（p37）。
  • 一元加操作符以加号（+）号表示，放在数值前面，无任何反应。放在非数值前面，会像Number()转型函数一元对这个值进行改变。
  • 一元减操作符以减号（-）号表示，放在数值前面，用来表示负数。放在非数值前面，会像Number()转型函数一元对这个值进行改变，然后再将得到的数值转换为负数。

17.位操作符用于再最基本的层次上，即按照内存中表示数值的位来操作数值。

ECMAScript中所有数值都是以IEEE-754 64位格式存储，但位操作符并不直接操作64位的值，而是先将64位的值转换成32位的整数，然后执行操作，最后再将结果转换回64位。对于有符号的整数，32位中前31位表示整数的值，最后一位位符号位，0表示正数，1表示负数。正数以纯二进制格式存储，没有用到的位用0填充。例如数值18的二进制表示是00000000000000000000000000010010。负数同样以二进制码储存，但使用的格式是二进制补码（先求整数的二进制码，然后0变1，1变0，最后再加1）

  • 按位非（NOT）

    • 由一个波浪线（~）表示，结果就是返回数值的反码（0变1，1变0）

      var num1 = 25;      //00000000000000000000000000011001
var num2 = ~num1;   //11111111111111111111111111100110
alert(num2);        //-26

  • 按位与（AND）

    • 由一个和号字符（&）表示，它有两个操作数，两个数值对应位都是1才返回1，任何一位是0就返回0。

      var result = 25 & 3;
alert(result);     //1

  • 按位或（OR）

    • 由一个竖线符号（|）表示，它有两个操作数，有一个位是1的情况就返回1，只有两个位都是0的情况下才返回0。

      var result = 25 | 3;
alert(result);     //27

  • 按位异或（XOR）

    • 由一个插入符号（^）表示，它有两个操作数，两个数值位上只有一个1时才返回1，如果对应两位都是1或都是0，则返回0。

      var result = 25 ^ 3;
alert(result);     //26

  • 左移

    • 由两个小于号（<<）表示，将数值的所有位向左移动指定的位数，左移不会影响符号位，右侧多出的空位用0来填充。
    var oldValue = 2;                //等于二进制的10
var newValue = oldValue << 5;    //等于二进制的1000000，十进制的64

  • 有符号的右移

    • 有符号的右移由两个大于号（>>）表示，将数值向右移动，但保留符号位，左侧出现空位按照符号位的值进行填充。

  • 无符号的右移

    • 无符号的右移由三个大于号（>>>）表示，将数值的32位都向右移动，左侧出现的空位用0来填充。因此对于正数来说，无符号右移与有符号右移结果是相同的，但对于负数来说，由于符号位移动走了，无符号右移操作符会把负数的二进制码当成正数的二进制码，导致无符号右移后结果非常大。

      var oldValue = -64;                //等于二进制的11111111111111111111111111000000
var newValue = oldValue >>> 5;    //等于十进制的134217726

18.布尔操作符一共有三个：非（NOT）、与（AND）和或（OR）

  • 逻辑非—由一个感叹号（！）表示，无论这个值是什么数据类型，这个操作符都会返回一个布尔值。首先将它的操作数转换为一个布尔值，然后对其求反。使用两个逻辑非操作符，就相当于Boolean()转型函数的行为。

    alert(!false);      //true
alert(!!"blue");    //true

  • 逻辑与—由两个和好（&&）表示，两个操作数为真，才为真。

    • 逻辑与操作可以应用于任何类型的操作数，不仅仅是布尔值。规则（p45）。
    • 逻辑与操作属于短路操作，即如果第一个操作数能决定结果，那么就不会再对第二个操作数求值。
    var result = true && false;  //false

  • 逻辑或—由两个竖线符号（||）表示，一个为真，就为真。

    • 与逻辑与类似，如果有一个操作数不是布尔值，逻辑或也不一定返回布尔值，常用于设置默认值。规则（p46）。
    • 逻辑或操作也属于短路操作。

19.运算符有乘、除、求模(%)、加法、减法，再操作数为非数值的情况下会执行自动的类型转换。（p47）

20.关系操作符有小于、大于、小于等于、大于等于，操作数是非数值时，也要进行数据转换或完成某写奇怪的操作。（p50）

21.相等操作符：

  • 相等（==）和不相等（!=）：先转换操作数(通常为强制转型)，然后再进行比较它们的相等性。（转换规则p51）
  • 全等（===）和不全等（!==）:不转换直接进行比较他们的相等性。

22.条件操作符：

variable = boolean_expression ? true_value : false_value;

23.简单的赋值操作符由等于号（=）表示，作用就是将右边的值赋给左侧的变量。每个主要算数运算操作符都有对应的复合赋值操作符（*=、/=、%=、+=、-=、<<=、>>=、>>>=）

24.逗号操作符多用于声明多个变量，还可用于赋值，但再用于赋值时，逗号操作符总是会返回表达式中最后一项（不常见）。

3.6语句

25.if语句，业界普遍推崇的最佳实践是始终使用代码块，即使执行的只有一行代码。

if (condition) statement1 else statement2

26.do-while语句是一种后测试循环语句，循环体内代码至少执行一次。

do {
    statement
} while (experssion)

27.while语句属于前测试循环语句，不符合条件，循环体内的代码可能永远不会执行

while(experssion) statement

28.for语句也是一种前测试循环语句，但它具有再执行循环之前初始化变量和定义循环后要执行的代码的能力。

for (initialization; expression; post-loop-expression) statement

29.for-in语句是一种精准的迭代语句，可以用来枚举对象的属性。ECMAScript对象的属性是没有顺序的，因此输出的属性名的顺序是不可预测的。如果要迭代的对象的变量值为null或undefined，for-in语句会报错，虽然ECMAScript5更正了这一行为，但为了保证兼容性，使用for-in语句之前，先检查该对象的值是不是null或undefined。

for (property in expression) statement

30.使用label语句可以在代码中添加标签，以便将来使用。加标签的语句一般都要与for语句等循环语句配合使用。标签语法如下

label:statement
//例子
start: for (var i = 0; i < count; i++) {
    alert(i)
}

31.break和continue语句用于在循环中精确地控制代码执行，其中break语句会立即退出循环，而continue语句虽然也是立即退出循环，但退出循环后会从循环的顶部继续执行。

braek和continue语句都可以与label语句联合使用，从而返回代码中的特定位置。如果使用label语句，一定要使用描述性的标签，同时不要嵌套过多的循环。

var num = 0;
outermost:
for(var i = 0; i < 10; i++) {
    for(var j = 0; j < 10; j++) {
        if(i == 5; j < 10; j++) {
            break outermost;   
        }
        num++;
    }
}
alert(num);    //55

break语句不仅会退出内部的for语句（即使用变量j的循环）,而且也会退出外部的for语句（即使用变量为i的循环）。

var num = 0;
outermost:
for(var i = 0; i < 10; i++) {
    for(var j = 0; j < 10; j++) {
        if(i == 5; j < 10; j++) {
            continue outermost;
        }
        num++;
    }
}

alert(num);    //95

continue语句会强制继续执行循环——退出内部循环，执行外部循环。

32.switch语句

  • case的含义是：“如果表达式等于这个值，则执行后面的语句（statement）”
  • break关键字会导致代码执行流跳出switch语句。
  • default关键字用于在表达式不匹配前面任何一种情形时，执行机动代码。
  • 通过为每个case后面都添加一个break语句，就可以避免同时执行多个case代码的情况。
  • 可以在switch语句中使用任何数据类型，无论是字符串，还是对象都可以。
  • switch语句在比较值时使用的是全等操作符，因此不会发生类型转换（即“10”不等于10）
switch (expression) {
    case value: statement
        break;
    case value: statement
        break;
    //下面合并两句，要添加注释，说明你是有意省略了break关键字
    case value:
    case value: statement
        break;
    default: statement
}

3.7函数

33.ECMAScript中函数在定义时不必指定是否返回值。函数会在执行完return语句之后停止并立即退出，位于return后的任何代码永远不会执行。return语句也可以不带有任何返回值，停止执行后将返回undefinde值。

34.ECMAScript函数不介意传递进多少个参数，也不在乎传进来的参数是什么数据类型。原因是ECMAScript中参数在内部是用一个数组来表示的，函数接收到的始终都是这个数组，而不关心数组中包含哪些参数，函数体内可以通过arguments对象来访问这个参数数组。

arguments对象只是与数组类似（它并不是Array的实例），可以使用方括号访问它每一个元素，length属性来确定传递进来多少个参数。因此函数可以不显示地使用命名参数，在代码中直接以arguments[0]来代表，传递进来的第一个参数。可以利用这一点让函数能够接收任意个参数并分别实现适当的功能。arguments对象可以与命名参数一起使用，他的值永远与对应命名参数的值保持同步，这并不是说读取这两个值会访问相同的内存空间，他们的内存空间是独立的，但它们的值会同步。没有传递值的命名参数将自动赋予undefined的值。

35.ECMAScript函数不能像传统意义上那样实现重载，如果定义了两个名字相同的函数，则改名字只属于后定义的函数。

变量、作用域和内存问题

4.1基本类型和引用类型的值

36.变量分为基本类型和引用类型

  • 基本类型—值在内存中占据固定大小的空间，因此被保存在栈内存中。从一个变量向另一个变量复制基本类型的值，会创建这个值的一个副本，两者没有任何关系。
  • 引用类型—值是对象，保存在堆内存中(无序，空间大小可变)。包含引用类型值的变量实际上包含的并不是对象本身，而是一个指向该对象的指针<指针保持在栈内存中>。从一个变量向另一个变量复制引用类型的值，复制的其实是指针，因此两个变量最终都指向一个对象。

37.ECMAScript中所有函数的参数都是按值传递。传递参数：指的是将函数外的变量传递给函数当变量；要先判断传入的这个变量是什么类型的变量？若是基本类型，则传进来的变量与函数外部的变量就毫无关系，只是值相等。若是引用类型，则传进来的变量与函数外部的变量都只是一个指针，指向同一个对象。

//传递进来的变量是基本类型的变量
var num = 10;
function fn(num2) {
    num2 +=10;
    return num2;
}
var num3 = fn(num);
alert(num);     //10
alert(num3);    //20
--------------------------------------
//传递进来的变量是引用类型的变量
function fn(obj) {
    obj.name = "dilei";
}
var person = new Object();
person.name = "landmine";
fn(person);
alert(person.name);     //dilei

38.基本类型与基本类型的比较，值和类型相等就相等。

39.引用类型与引用类型的比较，只有地址指向同一个引用才

var aa = {
    age:18;
    score:4;
};
var aa = bb;
console.log(aa === bb)    //true

var cc = {
    age:18;
    score:4;
};
console.log(aa === cc)    //false

4.2作用域

40.作用域（又称执行环境）分为全局作用域和局部作用域，可理解为（作用范围，变量生命周期）

  • 全局作用域：其变量叫全局变量，到处都能访问到。整个程序结束才销毁。
  • 局部作用域：通常指函数作用域，其变量叫局部变量，只能在函数内访问。函数调用完成后就会立即销毁。

每个作用域都有一个与之关联的变量对象，作用域中定义的所有变量和函数都保存在这个对象中。全局作用域是最外围的一个作用域，在web浏览器中，全局作用域被认为是window对象。因此所有全局变量和函数都是作为window对象的属性和方法创建的。每个函数都有自己的局部作用域。每次进入一个新的作用域时，都会创建一个用于搜索变量和函数的作用域链。作用域链的用途是保证按特定规则有序访问变量和函数。作用域的前端，始终是当前执行代码所在作用域的变量对象；作用域链中下一个变量对象来自下一个包含关系的作用域；再下一个变量对象则来自于下一个包含关系的作用域。

每个作用域都可以向上搜索作用域链，以查询变量和函数名。但任何作用域都不能通过向下搜索作用域链而进入另一个作用域。

41.在某个环境中为了读取或写入而引用一个标识符时，必须通过搜索来确定该标识符实际代表什么。搜索过程从作用域链的前端开始，一直追溯到全局环境的变量对象。如果局部环境中存在着同名标识符，就不会使用父环境中的标识符。

42.javascript解析机制（预解析>>逐行解读代码）

  • 预解析—现在不同作用域里进行预解析
    • ①在window里，查找所有var,将其变量赋值为undefined，再查找function,将其直接拿过来（非赋值）。
    • ②再到局部作用域里，同样步骤。
    • 遇到冲突解决方法
      • 变量名与变量名一样，不影响，都为undefined。
      • 变量名与函数名一样，保留函数。
      • 函数名与函数名一样，保留最后一个函数
      • 注：在一些老版本浏览器无法预解析代码块里的函数，一次在块里不要定义函数，要么放在全局里，要么放在变量里。
  • 逐行解读代码
    • 将变量的值，替换掉undefined。
    • 读到函数时，就跳过。（预解析时已经声明了）
  • 注意：解析是分标签进行的，上一个标签解析完后，再进行下一个标签

报错

43.javascript没有块级作用域，因此注意由for循环创建的i变量即使在循环结束后也不会销毁，依旧会存在于循环外部的作用域中。

44.使用var声明的变量会自动添加到最接近的环境中，没有使用var声明，该变量会自动被添加到全局环境。严格模式下，初始化未经声明的变量会导致错误。

4.3垃圾收集

45.javascript是一门具有垃圾回收机制的编程语言，开发人员不用担心内存分配和回收的问题。原理：找出没用数据>>打上标记>>释放其内存>>周期性执行。

标识无用数据的策略：

  • 标记清除：垃圾收集器在运行时，会把储存在内存中的所有变量打上标记。然后去掉作用域中的变量 以及被这些作用域变量所引用的变量。
  • 引用计数：跟踪记录每个数据的应用次数。被变量应用一次就加1，引用的变量更新了就减1，次数为0就代表没有变量引用它了就被标记。存在循环引用的问题，需要手动设置为null。

46.虽然javascript有垃圾回收机制，但分配给web浏览器的内存是要少于桌面应用程序的。因此内存还是省着点用比较好。

优化内存占用的最佳方式，就是为执行中的代码只保存必要的数据。一旦数据不再有用，最好通过将其设置为null来释放其引用。适合于大多数全局变量，因为局部变量在离开作用域时会被自动解除引用。而全局变量会在整个程序结束以后才被解除。

引用类型

47.引用类型的值(对象)是引用类型的一个实例。在ECMAScript中，引用类型是一种数据结构(字典，键值对)用于将数据和功能组织在一起。

object类型

48.对象是某个特定引用类型的实例。创建Object实例的方式有两种；

  • 使用new操作符后跟object构造函数，构造函数本身就是一个函数，只不过该构造函数是出于创建新对象的目的而定义的。

    var person = new object();
person.name = "jinhao";

  • 使用对象字面量表示法，用逗号来分隔不同的属性。这种方式简化了创建包含大量属性的对象的过程。并不会调用object构造函数。

    var person = {
    name : "jinhao",
    age : 21
},

最好的做法是对那些必须值使用命名参数，而使用对象字面量来封装多个可选参数。

49.访问对象属性的方法；

  • 点表示法

    alert(person.name)

  • 方括号表示法，将要访问的属性以字符串的形式放在方括号中。针对的范围更广，可以通过变量来访问属性，若属性名中包含会导致语法错误的字符串，属性名使用关键字或保留字等都可以使用方括号表示法

    alert(person["name"])

通常，尽量用点表示法，会出错再用方括号表示法。

Array类型

50.ECMAScript数组与其他语言中的数组都是数据的有序列表，但ECMAScript数组的每一项可以保存任何类型的数据。其数组大小是可以动态调整的，即可以随着数据的添加自动增长以容纳新增数据。

51.创建数组的基本方式由两种；

  • 使用Array构造函数

    var colors = new Array();
var colors = new Array(20);    //可以给构造函数传递数量，改数量会自动变成length属性的值
var colors = new Array("Greg");  //传递的不是数值，则只会创建包含那个值的只有一项的数组
var colors = new Array(“red”,"blue","green");   //也可以传递数组中应该包含的项

  • 使用数组字面量表示法，多个数组项用逗号隔开。不会调用Array构造函数。

    var colors = ["red", "blue", "green"];

52.读取和设置数组的值，要使用方括号并提供相应值基于0的数字索引。

var colors = ["red", "blue", "green"];
alert(colors[0]);      //显示第一项的值 red
colors[0] = "black";   //修改第一项的值为 black

如果索引值大于数组中的项数，数组就会自动增加到该索引值加1的长度（因为是从0开始的，数组最后一个值索引为3，实际长度就是4），新增后，不存在的项会返回undefined。

53.数组的length属性不是只读的，还可以通过这个属性，从数组的末尾移除项或向数组中添加新项。

//从末尾移除项
var colors = ["red", "blue", "green"];
colors.length = 2;
alert(color[2])    //undefined

//从末尾添加项
var colors = ["red", "blue", "green"];
colors.length = 4;    //多出来的项，返回undefined
alert(color[3])       //undefined

//从末尾添加项
var colors = ["red", "blue", "green"];
colors[colors.length] = "black";   //最后一个位置添加black

54.用instanceof操作符检测数组，有个问题，它假定只有一个全局执行环境。而Array.isArray()方法，可以判断值是不是数组且不管他是在哪个全局执行环境中创建的,用法如下；

if (Array.isArray(value)) {
    //对数组执行某些操作
}

valueOf()、toString()、toLocaleString()、join()

55.如前所述(p35)，所有对象都具有toLocaleString()、toString()和valueOf()方法，数组对象继承的这3个方法，默认情况下都会以逗号分隔的字符串的形式返回数组项。

  • 数组对象调用valueOf()，会返回数组本身。
  • 数组对象调用toString()，会返回有数组中每个值的字符串形式拼接而成的一个以逗号分隔的字符串。实际上，为了创建这个字符串会调用数组每一项的toString()方法。
  • 数组对象调用toLocaleString()，也会创建一个数组值的以逗号分隔的字符串，与前面两个不同之处在于，这一次为了取得每一项的值，调用的是每一项的toLocaleString()方法。 
var colors = ["red", "blue", "green"];
alert(colors.toString());           //red,blue,green
alert(colors.toLocaleString());     //red,blue,green
//由于alert（）要接收字符串，它会在后台调用toString方法，因此结果其他两个差不多
alert(colors.valueOf());            //red,blue,green

使用join()方法，可以使用不同的分隔符，来构建这个字符串。其只接受一个参数：用作分隔的字符串。

var colors = ["red", "blue", "green"];
alert(colors.join("||"));           //red||blue||green

push()、pop()

56.ECMAScript数组也提供了一种让数组的行为类似于其他数据结构的方法，ECMAScript为数组专门提供了push()和pop()方法以实现类似栈（后进先出）的行为。

  • push()方法可以接收任意数量的参数，把他们逐个添加到数组末尾，并返回修改后数组的长度。
  • pop()方法则从数组末尾移除最后一项，减少数组的length值，然后返回移除的项。
var colors = new Array();                 //创建一个数组
var count = colors.push("red", "green");  //推入两项
alert(count);     //2

var item = color.pop();                   //取得最后一项
alert(item);      //"green"
alert(colors.length);  //1

shift()、push()、unshift()

57.结合使用shift()和push()方法，以实现类似队列（先进先出）的行为。

  • shift()能够移除数组中的第一项并返回该项。
var colors = new Array();                 //创建一个数组
var count = colors.push("red", "green");  //推入两项
alert（count）  //2

var item = color.shift();                 //取得第一项
alert(item);   //"red"

同时使用unshitf()和pop()方法，可以从相反的方向模拟队列。

  • unshift()他能在数组前端添加任意个项并返回新数组长度。
var colors = new Array();                    //创建一个数组
var count = colors.unshift("red", "green");  //推入两项
alert（count）  //2

var item = colors.pop();                     //取得最后一项
alert（item）   //"green"

reverse()、sort()

58.数组中有两个用来重排序的方法：reverse()和sort()

  • reverse()会反转数组项的顺序

  • sort()按升序排列数组项，会调用每个数组项的toString()转型方法，然后再进行比较（如5和10比较，结果“10”会在“5”前面）。这种排序方法在很多情况下都不是最佳方案，因此sort()可以接受一个比较函数作为参数。

    • 比较函数接受两个参数，比较函数通过返回一个小于零、等于零或大于零的值来影响排序结果。以下是一个简单的比较函数。

      function compare(value1, value2) {
    if (value1 < value2) {
        return -1;
    } else if(value1 > value2) {
        teturn 1;
    } else {
        return 0;
    }
}

  • 对于数值类型或者valueOf()方法返回数值类型的对象类型，可以使用一个更简单的比较函数。

    function compare(value1, value2) {
    return value2 - value1;
}

concat()、slice()、splice()

59.concat()方法可以基于当前数组中的所有项创建一个新数组。这个方法会先创建当前数组的一个副本，然后将接收到的参数添加到这个副本的末尾，最后返回新构建的数组。参数如果是数组，则会将数组里的每一项都添加进当前数组。

var colors = ["red", "green", "blue"];
var colors2 = colors.concat("yellow", ["black", "brown"]);
alert(colors2);    //red,green,blue,yellow,black,brown

60.slice()方法能够基于当前数组中的一或多个创建一个新数组。slice()可以接受一到两个参数，只有一个参数的话，会返回参数指定位置开始到当前数组末尾的所有项；如果是两个参数的话，返回起始和结束位置之间的项，但不包括结束位置的项。

var colors = ["red", "green", "blue", "yellow", "purple"];
var colors2 = colors.slice(1);
var colors3 = colors.slice(1, 4);
alert(colors2);    //green,blue,yellow,purple
alert(colors2);    //green,blue,yellow

如果有一个参数是负数，则用数组长度加上该数来确定相应的位置。（如，数组长度为5，slice(-2,-1)与slice(3,4)得到的结果相同）如果相加后，起始位置大于结束位置则返回空数组。

61.splice()方法主要用途是想数组中部插入项，使用这种方法的方式有如下3种；

  • 删除——可以删除任意数量的项，需指定2个参数：要删除的第一项的位置、要删除的项数。
  • 插入——可以向指定位置插入任意数量的项。需指定3个参数：起始位置、0、要插入的项(可任意多项)。
  • 替换——可以向指定位置插入任意数量的项，且同时删除任意数量的项。需指定3个参数：起始位置、要删除的项数、要插入的任意数量的项(不必与删除相等)
var colors = ["red", "green", "blue"];

var removed = colors.splice(0,1)       //删除第一项
alert(colors);     //green,blue
alert(removed);    //red,返回的数组中只有一项

removed = colors.splice(1, 0, "yellow", "orange"); //从位置1插入两项
alert(colors);     //green,yellow,orange,blue
alert(removed);    //返回的是一个空数组

removed = colors.splice(1, 1, "red", "purple"); //插入两项，删除一项
alert(colors);     //green,red,purple,orange,blue
alert(removed);    //yellow,返回的数组中只有一项

indexOf()、lastIndexOf()

62.ECMAScript5为数组实例添加了两个位置方法：indexOf()和lastIndexOf()，都接收两个参数：要查找的项、查找起点位置的索引(可选)。indexOf()从头开始查找，lastIndexOf()从尾开始查找。这两个方法都要返回要查找的项在数组中的位置，实在没找到的情况就返回-1。比较的时候会使用全等操作符。

var colors = ["red", "green", "blue", "yellow", "purple"];
alert(colors.indexOf("green"));         //1
alert(colors.lastIndexOf("green"));     //3

every()、filter()、forEach()、map()、some()

63.ECMAScript5为数组定义了5个迭代方法；每个方法都接收两个参数：要在每一项上运行的函数、运行在改函数的作用域对象(可选，影响this的值)。传入这些方法的函数会接收三个参数：数组项的值item、该项在数组中的位置index、数组本身array。以下方法都不会修改数组中的包含的值。

  • every()——数组的每一项都会在函数里进行处理，如果全部都返回true，则返回true。
  • filter()——数组的每一项都会在函数里进行处理，返回true的项组成的数组。
  • forEach()——数组的每一项都会在函数里进行处理，但不返回值。
  • map()——数组的每一项都会在函数里进行处理，将每次调用的结果组成数组。
  • some()——数组的每一项都会在函数里进行处理，如果该函数对任一项返回true，则返回true。
//返回一个所有数值都大于2的数组
var numbers = [1,2,3,4,5,6];
var filterResult = numbers,filter(function(item,index,array){
    teturn (item > 2);
});
alert(filterResult)  //[3,4,5,6]

reduce()、reduceRight()

64.ECMAScript5还新增了两个归并数组的方法：reduce()和reduceRigth()，这两个方法都会送代数组的所有项，然后返回一个最终返回的值。reduce()从头开始，reduceRight()从右开始。两个方法都接收两个参数：一个在每一项上调用的函数、作为归并基础的初始值(可选)。传入的函数接收4个参数：前一个值prev、当前值cur、项的索引index、数组对象array，该函数返回的任何值都会作为第一个参数自动传递给下一项。

var values = [1,2,3,4,5];
var sum = values.reduce(function(prev,cur,index,array){
    retuen prev + cur;
});
alert(sum); //15

第一次执行调用函数，prev是1，cur是2；第二次，prev是3(1+2的结果)，cur是3(数组第三项)。整个过程会把数组中的每一项都访问一遍。
