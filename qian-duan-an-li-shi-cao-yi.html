创建文件结构

在项目开始之前，创建好一个文件夹，里面的内容如下

分析页面结构，将结构写下来

<!-- 页面的顶部条状物 -->
<section id="top"></section>
<!-- 页面的头部 -->
<header></header>
<!-- 页面的导航 -->
<nav></nav>
<!-- 页面的banner部分 -->
<section id="banner"></section>
<!-- 页面的明星机型部分 -->
<section id="star"></section>
<!-- 页面的精选配件部分 -->
<section id="accessory"></section>
<!-- 页面的搜索欧珀部分 -->
<section id="world"></section>
<!-- 页面的服务部分 -->
<section id="serve"></section>
<!-- 页面的售后服务部分 -->
<section id="after_sale"></section>
<!-- 页面的底部 -->
<footer></footer>

给页面结构初始化样式，将默认样式清除掉

/* 初始化样式 */
* {
    margin: 0;
    padding: 0;
}
ol,ul {
    list-style-type: none;
}
a {
    color: #666;
    text-decoration: none;
}
.left {
    float: left;
}
.right {
    float: right;
}

测试一下，分块是否正确

/* 测试 */
#top,header,nav,#banner,#star,#accessory,
#world,#serve,#after_sale,footer {
    height: 100px;
    width: 100%;
    border: 1px solid #000;
}

完善结构框架（宽度、居中等）

<!-- 页面的顶部条状物 -->
<section id="top">
    <div class="container"></div>
</section>
<!-- 页面的头部 -->
<header>
    <div class = "container"></div>
</header>
<!-- 页面的导航 -->
<nav>
    <div class = "container"></div>
</nav>
<!-- 页面的banner部分 -->
<section id="banner">
    <div class = "container"></div>
</section>
<!-- 页面的明星机型部分 -->
<section id="star">
    <div class = "container"></div>
</section>
<!-- 页面的精选配件部分 -->
<section id="accessory">
    <div class = "container"></div>
</section>
<!-- 页面的搜索欧珀部分 -->
<section id="world">
    <div class = "container"></div>
</section>
<!-- 页面的服务部分 -->
<section id="serve">
    <div class = "container"></div>
</section>
<!-- 页面的售后服务部分 -->
<section id="after_sale">
    <div class = "container"></div>
</section>
<!-- 页面的底部 -->
<footer>
    <div class = "container"></div>
</footer>
/* 内容居中显示，宽度为1280px */
.container {
    width: 1280px;
    border: 1px solid #f00;
    margin: 0px auto;
}

每个部分挨着做

头部

1.分析：怎样分块，块与快之间怎么嵌套

2.显示图片的特定位置

如图，想只显示图片中的微博图标。可在PS软件中，测一下该小图标的宽高，和离原图左上角的距离是多少。然后设置CSS样式即可。

若改图片不是放在块元素里，而是放在行级元素里，则是不能设置宽度和高度的，需要用到display属性设置为block或inline-block。垂直居中用vertical-aline: middle。

  • inline元素特点(,)

    不单独占用一行，width、height不可设置

  • block元素特点(

    ,

    ,

    ,
      ,
        ,
      • ，,)

        单独占一行，width、height、margin和padding都可以设置

      • inline-block元素特点()

        不用单独占一行，width、height、margin和padding都可设置

        • 3.块级元素之间，怎样一排显示，且有一定间隔

        4.导航栏，各元素之间存在竖杠分隔
        • 方法1：设置CSS属性，给每个元素的一边设置边框和内边距，最后一个元素取消边框。
        header li a {
    color: #b2b2b2;
    font-size: 14px;
    border-right: 1px solid #b2b2b2;  //给每个元素的右边设置边框
    padding: 5px 15px 0px;  //设置内边距
}
header .shop_car a {
    border: none;    //最后一个元素单独设置，取消边框
}
        • 方法2：手动在每个元素旁边加字符(|) &sbsp;代表空格
